# parasail-intro

A lightweight Node.js app that serves a one-page memo for an AI-powered physical-world distribution business.

The current page reframes the opportunity as a **system of record + execution layer for physical-world distribution**, starting with beverage as the initial wedge.

## Development

- `npm start` – start a local server at http://localhost:3000
- `npm test` – run a basic test to ensure the server returns the memo page
